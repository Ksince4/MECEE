# MECEE

Application de calcul MECEE (Windows).

## Téléchargement

➡️ **[Télécharger la dernière version](../../releases/latest)**

1. Va dans **Releases** (colonne de droite) ou clique le lien ci-dessus.
2. Télécharge le fichier **`MECEE.exe`**.
3. Lance-le. Une fenêtre de connexion s'ouvre : entre les identifiants
   fournis par l'administrateur.

> ⚠️ Une connexion Internet est nécessaire au démarrage (vérification du compte).

## Support

Pour obtenir un accès ou en cas de problème de connexion, contacte l'administrateur.
